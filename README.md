HeelsFetishism-Deployment
=========================

SaltStack States for [HeelsFetishism](http://heelsfetishism.com/)


## States

* Django
* PostgreSQL
* nginx
* uWSGI
* Memcached
* imgSeek
* Node.js


## Usage

Place your pillar settings in `/srv/pillar/settings.sls`

``` bash
# on salt-master server
$ sudo salt '*' state.highstate
```
